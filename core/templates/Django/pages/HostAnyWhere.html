Host Your Django Website on PythonAnywhere

A step-by-step guide to deploying your Django application effortlessly.

Get Started with PythonAnywhere

Step 1: Create an Account

Go to the PythonAnywhere website and sign up for an account. The free tier is sufficient for small projects.

Step 2: Upload Your Django Project

Log in, navigate to the "Files" tab, and upload your Django project files. Alternatively, use the Bash console to clone your project from Git.

git clone https://github.com/yourusername/yourproject.git

Step 3: Set Up a Virtual Environment

Start a new Bash console and create a virtual environment:

mkvirtualenv myenv --python=python3.8
pip install django
pip install -r requirements.txt

Replace myenv with your virtual environment name and adjust the Python version if needed.

Step 4: Configure Your Web App

Navigate to the "Web" tab, click "Add a new web app", choose "Manual configuration", and select the appropriate Python version. Specify your project's working directory and the path to the WSGI file. Add the following lines to your WSGI file:

import os
import sys

project_home = u'/home/yourusername/yourproject'
if project_home not in sys.path:
    sys.path = [project_home] + sys.path

os.environ['DJANGO_SETTINGS_MODULE'] = 'yourproject.settings'

from django.core.wsgi import get_wsgi_application
application = get_wsgi_application()

Step 5: Configure Your Static Files

Click on the "Web" tab, scroll to the bottom, and click on the URL to add the static files. Then add the complete URL of your static files:

STATIC_URL = '/static/'
STATIC_ROOT = os.path.join(BASE_DIR, 'staticfiles')

Step 6: Set Up Your Database

For databases like PostgreSQL or MySQL, set up the database on PythonAnywhere and update your Django settings. Find database setup instructions under the "Databases" tab.

Step 7: Collect Static Files

In your Bash console, run:

python manage.py collectstatic

Step 8: Apply Migrations

Run the following commands to apply your database migrations:

python manage.py makemigrations
python manage.py migrate

Step 9: Test Your Application

Navigate to the URL provided by PythonAnywhere to see your live Django application. Test all features to ensure everything works as expected.

Conclusion

Deploying your Django website on PythonAnywhere is straightforward. By following these steps, you can quickly get your site up and running in a reliable and scalable environment.
